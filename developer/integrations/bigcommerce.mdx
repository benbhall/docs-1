<Steps>
<Step title="Run the CLI">
To get started with the BigCommerce Catalyst example, run the following command. This will clone the [Makeswift BigCommerce catalyst example](https://github.com/makeswift/makeswift/tree/main/examples/bigcommerce-catalyst) from GitHub.

```bash
npx makeswift@latest init --template=bigcommerce-catalyst
```

</Step>

<Step title="Create a new site">
Choose your workspace, then create a new site.

</Step>
