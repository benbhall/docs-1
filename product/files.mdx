## Introduction

This guide provides an overview of the file management and sorting features available in Makeswift. These features are designed to help you efficiently organize, manage, and locate your files within the platform.

### File Management

Use the features available within the file manager to control the assets within your site.

#### Bulk Actions

You can perform actions on multiple files at once to streamline your workflow.

1. **Select Files:** Click the checkbox next to each file you want to select.
2. **Perform Actions:** Use the action buttons (delete, move, download) at the top of the file manager to perform bulk actions on the selected files.

#### Drag and Drop

Easily move files and folders within the file manager by using the drag-and-drop feature.

1. **Select a File or Folder:** Click and hold the file or folder you want to move.
2. **Drag to New Location:** Drag the selected item to the desired location within the file manager.
3. **Drop to Move:** Release the mouse button to drop the item into the new location.

#### Quick Preview

View a preview of your files without opening them fully to save time by hovering over a file and clicking the preview icon.

### Sorting Options

Organize your files using various sorting options to easily locate what you need. You can sort by **date**, alphabetically by **name**, or by **type** by clicking on the corresponding headers.

### Search Functionality

You can use the search feature to quickly find specific files in your library by either utilizing **Keywords** or **Filters**

#### Keyword Search

Type keywords into the search bar at the top of the file manager and press Enter.

#### Filters

Use the filter options to limit results by date, type, or other attributes.
